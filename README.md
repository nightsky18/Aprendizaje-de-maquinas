# Aprendizaje de Máquinas y Computación Evolutiva

Repositorio que contiene los **ejercicios, prácticas de clase y proyectos** desarrollados durante la asignatura **Aprendizaje de Máquinas y Computación Evolutiva**.

La materia hace parte de la **línea de profundización de la la carrera de Ingeniería Informática** en el **Politécnico Colombiano Jaime Isaza Cadavid**.

---

## Objetivo del repositorio

Este repositorio funciona como un espacio de organización y documentación del trabajo realizado durante el curso, incluyendo:

- Ejercicios prácticos vistos en clase  
- Implementaciones en Python  
- Análisis de datasets  
- Experimentos con algoritmos de aprendizaje automático  
- Cuadernos de trabajo en **Jupyter Notebook / Google Colab**

El objetivo es mantener un registro estructurado del proceso de aprendizaje y de los distintos experimentos realizados durante la asignatura.

---

## Contenido actual

Actualmente el repositorio contiene los siguientes cuadernos:

### 📓 Cuaderno_1_Iris.ipynb
Exploración inicial de aprendizaje automático utilizando el dataset clásico **Iris**.  
Incluye análisis de datos y primeras aproximaciones a modelos de clasificación.

### 📓 Cuaderno_2_Wine.ipynb
Trabajo con el dataset **Wine**, enfocado en análisis de características y experimentación con algoritmos de clasificación.

---

## Tecnologías utilizadas

Las prácticas se desarrollan principalmente con:

- Python  
- Jupyter Notebook  
- Google Colab  
- NumPy  
- Pandas  
- Matplotlib  
- Scikit-learn  

Estas herramientas permiten trabajar con datos, visualizarlos y construir modelos de aprendizaje automático.

A medida que avance el curso, se agregarán nuevos cuadernos con ejercicios, experimentos y proyectos relacionados con **aprendizaje automático y computación evolutiva**.
